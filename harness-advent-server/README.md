# Сервер Harness Advent

Сервер — будущая серверная часть Harness Advent на Kotlin и Ktor. Он будет предоставлять API для заданий, проектов, RAG, оркестрации агентов, skills и контролируемых интеграций.

На текущем этапе модуль содержит только Ktor-заготовку. Он не реализует заявленные сценарии и не должен восприниматься как готовый API.

## Технологии

- Kotlin, JVM 21;
- Ktor;
- Koin для внедрения зависимостей;
- Exposed для доступа к данным;
- kotlinx.serialization для JSON-контрактов.

Архитектурные границы и правила безопасности описаны в документации корня проекта: [архитектура](../docs/architecture.md), [безопасность](../docs/security.md), [инструкции Codex](../AGENTS.md).

## Запуск

```bash
./gradlew test
./gradlew run
```

Для полной базовой проверки:

```bash
./gradlew test build
```

Локальные настройки и секреты не добавляй в репозиторий. При появлении конфигурации добавляй только безопасный `.env.example`, а реальные значения храни в окружении или игнорируемом `.env`.
